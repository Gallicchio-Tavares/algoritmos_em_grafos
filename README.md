# Trabalho de Algoritmos em Grafos

## Rodar o projeto

1. Crie a venv (Ambiente virtual):

   ``` bash
   python -m venv venv
   ```

2. Ative a venv

   ``` bash
   venv\Scripts\activate
   ```

3. Instale as bibliotecas

   ``` bash
   pip install -r requirements.txt
   ```

4. Execute `python main.py` para executar o c√≥digo:

   ``` bash
   python main.py
   ```
